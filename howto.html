üì• Input Data Format

The Comics server expects an Excel file with at least two sheets:

  1. Association Sheet
    Must contain the following column headers:
    - TRAITID1, TRAITID2 (mandatory)
    - Optional: PVALUE, BETA or COR
    If missing, default values will be used: PVALUE = 1 and BETA = 1.
  2. Annotation Sheet
    Must contain:
    - TRAITID, SHORTNAME, PLAT
    PLAT indicates the platform (e.g., OLINK, GWAS, EWAS, MBS) and defines node shape and color.

Optional: A Platform Style Sheet to define custom shape and color for each platform. If not provided, users can select them during import.

Note: Multiple association sheets are allowed. Only one annotation sheet is permitted.

üß≠ Basic Navigation

Start from the Home page. To customize it, go to Import Data > Upload Custom Home Page (HTML or Markdown).

üì§ Upload Data

Use the Import Data tab to upload your Excel file. A summary will display:

  • Total number of associations
  • Number of traits
  • Platforms identified

‚öôÔ∏è Customize Network

Users can:

  1. Assign shapes and colors to platforms.
  2. Select/deselect sheets for processing.

Clicking "Process" will populate the Table tab and enable network visualization.

üåê Network Interaction

  • Adjust "Max Nodes" dropdown (e.g., select 1 to show first-degree neighbors).
  • Hover over nodes/edges to view properties.
  • Drag nodes to rearrange the layout.
  • Click a node to show connected traits in the table below.

Search Tips: Use backspace to activate search in dropdowns. Use "Select Platform" to filter trait choices.

üîó Selecting Pairwise Associations

Under the Tables tab, browse and search associations across tabs like GWAS, EWAS, RWAS, MBH, GGM, GENO, CATA, STAT.

Click a row, then press "Focus network on selection" to view connected traits.

üíæ Exporting the Displayed Network

Use the Export tab to save the current network. To download the full Comics dataset, visit the GitHub page.
