# Multi-omics Server

A Shiny app for visualizing multi-omics networks.

## Installation
Install dependencies:
```R
install.packages(c("shiny", "shinydashboard", "readxl", "dplyr", "tidyr", "visNetwork", "DT", "shinyWidgets"))
```

## Usage
Run:
```R
shiny::runApp()
```

## Features
- Network visualization
- Data import from Excel
- Clear saved data option
